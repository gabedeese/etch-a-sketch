<h2 style="text-align: center;">This is a project for The Odin Project</h2>

***

<p>- Grid that works essentially like an etch-a-sketch</p>
<p>- Defaults to 16 x 16 but can be customized to contain blocks up to 100 x 100</p>
<p>- Has 3 modes</p>
<ul>
<li>Default = Fills blocks in black</li>
<li>RGB = Fills blocks in with random colors</li>
<li>Shade = Lightly shades in blocks more with each pass until they are fully black</li></ul>
